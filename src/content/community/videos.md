### Suspense உடன் server rendering-ஐ stream செய்தல் {/*streaming-server-rendering-with-suspense*/}

React 18, Suspense பயன்படுத்தி server-side rendering performance-க்கு மேம்பாடுகளையும் கொண்டுள்ளது.

Streaming server rendering, server-இல் React components-இலிருந்து HTML generate செய்து, அந்த HTML-ஐ உங்கள் users-க்கு stream செய்ய அனுமதிக்கிறது. React 18-இல், `Suspense` பயன்படுத்தி உங்கள் app-ஐ ஒன்றுக்கொன்று independent ஆன சிறிய units-ஆக பிரிக்கலாம்; app-ன் மீதியை block செய்யாமல் அவற்றை தனித்தனியாக stream செய்ய முடியும். இதனால் users உங்கள் content-ஐ விரைவில் காணவும், அதனுடன் மிகவும் வேகமாக interact செய்யத் தொடங்கவும் முடியும்.

Deep dive-க்கு, [Shaundai Person](https://twitter.com/shaundai)-ன் talk-ஐ இங்கே பார்க்கவும்:

<YouTubeIframe src="https://www.youtube.com/embed/pj5N-Khihgc" title="Streaming Server Rendering with Suspense" />

### முதல் React working group {/*the-first-react-working-group*/}

React 18-க்காக, experts, developers, library maintainers, மற்றும் educators அடங்கிய panel உடன் collaborate செய்ய எங்கள் முதல் Working Group-ஐ உருவாக்கினோம். ஒன்றாக, எங்கள் gradual adoption strategy-யை உருவாக்கியும் `useId`, `useSyncExternalStore`, மற்றும் `useInsertionEffect` போன்ற புதிய APIs-ஐ refine செய்தும் பணிபுரிந்தோம்.

இந்த பணியின் overview-க்கு, [Aakansha' Doshi](https://twitter.com/aakansha1216)-ன் talk-ஐ பார்க்கவும்:
