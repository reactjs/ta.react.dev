title: React சமூகம்
---

<Intro>

React-க்கு கோடிக்கணக்கான developers கொண்ட சமூகம் உள்ளது. நீங்கள் பங்கேற்கக்கூடிய சில React தொடர்பான சமூகங்களை இந்தப் பக்கத்தில் பட்டியலிட்டுள்ளோம்; கூடுதல் online மற்றும் நேரடி கற்றல் வளங்களுக்கு இந்தப் பிரிவில் உள்ள மற்ற பக்கங்களையும் பாருங்கள்.

</Intro>

## நடத்தை விதிகள் {/*code-of-conduct*/}

React சமூகங்களில் பங்கேற்பதற்கு முன், [எங்கள் நடத்தை விதிகளைப் படிக்கவும்.](https://github.com/facebook/react/blob/main/CODE_OF_CONDUCT.md) நாங்கள் [Contributor Covenant](https://www.contributor-covenant.org/)-ஐ ஏற்றுள்ளோம்; அனைத்து சமூக உறுப்பினர்களும் அதில் உள்ள வழிகாட்டுதல்களைப் பின்பற்ற வேண்டும் என எதிர்பார்க்கிறோம்.

## Stack Overflow {/*stack-overflow*/}

Code-level கேள்விகளை கேட்கவோ, குறிப்பிட்ட error-இல் சிக்கியிருக்கும்போதோ பயன்படுத்தப்படும் பிரபலமான forum Stack Overflow. **reactjs** tag கொண்ட [ஏற்கனவே உள்ள கேள்விகளை](https://stackoverflow.com/questions/tagged/reactjs) படியுங்கள் அல்லது [உங்கள் கேள்வியை நீங்களே கேளுங்கள்](https://stackoverflow.com/questions/ask?tags=reactjs)!

## பிரபலமான விவாத forums {/*popular-discussion-forums*/}

Best practices, application architecture, மற்றும் React-இன் எதிர்காலம் பற்றி விவாதிக்க ஏற்ற பல online forums உள்ளன. பதில் அளிக்கக்கூடிய code-level கேள்வி இருந்தால், பொதுவாக Stack Overflow சிறந்த இடமாக இருக்கும்.

ஒவ்வொரு சமூகத்திலும் ஆயிரக்கணக்கான React பயனர்கள் உள்ளனர்.

* [DEV's React community](https://dev.to/t/react)
* [Hashnode's React community](https://hashnode.com/n/reactjs)
* [Reactiflux online chat](https://discord.gg/reactiflux)
* [Reddit's React community](https://www.reddit.com/r/reactjs/)
